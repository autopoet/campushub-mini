# CampusHub Mini: Academic Buddy Spec

## 学习搭子匹配大厅设计指南 (V3.0 - 学术版)

### 1. 核心视觉哲学 (Aesthetics)

- **定位**：严肃但不死板，现代学霸风格。
- **色彩倾向**：增加莫兰迪蓝、森林绿等沉稳色调，减少高饱和粉色，体现学习的专注感。
- **排版原则**：
  - **分类标签**：全线统一为学术分类标签（如：`[ COMPETITION ]`）。
  - **极简大字报**：保持原有便利贴风格，但内容强制聚焦于学习目标。

### 2. 四大学术分类 (Core Categories)

1. **竞赛 (COMPETITION)**：建模、创新创业、挑战杯、程序设计等。
2. **考研 (POSTGRAD)**：跨校/同校考研、英语互督、政治背诵等。
3. **考公考编 (CIVIL)**：行测、申论、教资、编制考试等。
4. **日常学习 (DAILY)**：图书馆自习、四六级、选修课大作业等。

### 3. 社交准入机制 (Registration Strategy)

- **技能库更新**：
  - 默认展示强学习技能：`数学`、`英语`、`代码`、`法律`、`逻辑`、`申论`、`GPA管理`等。
- **渐进式补全**：
  - 注册期选填联系方式，执行“戳一戳学习搭子”时强制补全。

### 4. 名词术语规范 (Terminology)

- **中心词**：统一由“校园搭子”修改为 **“学习搭子”**。
- **交互动作**：点击动作 definition 为 **“戳他 (Poke)”**。
- **校友背书**：便利贴底部自动带上 `@ 学校`，利用校友圈建立学术信任。

### 5. 合规性设计 (Compliance)

- **学习导向**：通过限制分类，将原本泛社交的行为转化为垂直的学习互助，显著降低审核风险，提升教育类小程序的纯度。

### 6. 高级动效与交互规范 (UX Patterns)

- **物理感入场 (Paper Fall)**：首页贴纸采用 `Staggered Delay` 入场，模拟真实纸张飘落在墙面的重力感。
- **交叉倾斜布局 (Alternating Tilt)**：通过 `index % 2` 实现贴纸左右交互偏转（-1.5° / 1.5°），消除数字化列表的生硬感。
- **幽灵骨架屏 (Paper Ghost)**：加载态采用带有“心跳流光”和“倾斜角度”的卡片轮廓，与真实数据入场点位完美对齐。
- **iOS 级侧滑删除 (Swipe to Delete)**：
  - **手势优先级**：严格校验横纵位移比（`deltaX > deltaY`），防止垂直滚动时产生误触。
  - **状态闭合逻辑**：开启全局滚动收回机制，确保消息列表滚动时自动关闭已滑出的按钮。
  - **云端原子清理**：删除操作直接联动云开发数据库移除记录，实现“所见即所得”的数据新鲜度。
- **数据冗余存储 (Data Consistency)**：在发送信号时同步存储目标贴纸内容，确保原帖下架后互动记录依然能展示具体任务焦点。
